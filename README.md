# Flask MySQL Lab (Ultimate Docker Version)

Профессиональная сборка с использованием Docker, Makefile и переменных окружения.

## 🚀 Быстрый старт

Убедитесь, что у вас установлен `make`.

```bash
# 1. Собрать и запустить проект (в фоне)
make build up

# 2. Инициализация произойдет автоматически. Посмотреть логи:
make logs
```

## 🛠 Команды Makefile

- `make build` — сборка образов.
- `make up` — запуск контейнеров в фоне.
- `make down` — остановка контейнеров.
- `make restart` — перезапуск проекта.
- `make logs` — просмотр логов приложения.
- `make ps` — список запущенных сервисов.
- `make clean` — полная очистка (удаление томов БД и кэша).

## 🔐 Конфигурация
Все настройки (пароли, ключи) находятся в файле `.env`.
